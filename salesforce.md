# Salesforce Plugin Graph

This file is the future plugin-side ecosystem graph for Salesforce.

It should eventually describe:

- Salesforce platform capability areas
- when to route into each `sf-*` skill
- cross-skill workflows
- product and architecture decision guidance

- Integration and deployment

- AI and Agentforce
